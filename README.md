# Online Shopping Database System

## Overview

This project demonstrates the design and implementation of a hybrid database for an Online Shopping System using Microsoft SQL Server. The system combines relational database concepts with XML integration to manage customer information, products, orders, payments, deliveries, and reviews while maintaining data integrity through SQL constraints, triggers, views, and user-defined functions.

This project was developed as part of my academic learning to strengthen my understanding of advanced database concepts and practical SQL development.

---

## Objectives

- Design a normalized relational database for an online shopping platform.
- Implement business rules using SQL constraints and triggers.
- Integrate XML data with relational tables.
- Develop reusable SQL views and user-defined functions.
- Demonstrate data retrieval, modification, and management using SQL.
